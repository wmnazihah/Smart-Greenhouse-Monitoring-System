## Project Description
This project is an IoT Smart Greenhouse Monitoring System that collects sensor data, validates devices, stores telemetry data in MongoDB Atlas, and displays the data using MongoDB Charts.

## Technologies Used
- Node-RED
- MongoDB Atlas
- MongoDB Charts
- Docker
- Postman
